SCMs: là hệ thống quản lý mã nguồn (source code manage system)
- Phối hợp giữa các thành viên, giúp làm việc đồng thời trên các tệp tin
- Theo dõi được các tình trạng sửa lỗi và thực thi mã nguồn
- Quản lý tệp tin và kiểm soát phiên bản
VCS: Hệ thống quản lý phiên bản (Version code System)
- Khôi phục lại phiên bản cũ file và dữ án
- Xem lại ai là người tahy đổi và thay đổi những gì, có phát sinh sự cố hay không
- Khôi phục lại cả file bị xóa
-->Các phần mềm: Git, subversion, CVS, Bazaar,...
Các lệnh cần dùng
	-> lệnh git add . : đưa tất cả các file vào vùng nhớ 
	-> lệnh git commit -m "" : Nội dung tin nhắn, ghi nhận các thay đổi so với các phiên bản trước đó, và tạo ra 1 snapshot)
	-> lệnh git push origin main: đẩy mã nguồn từ local đến remote
